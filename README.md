# Topological Data Analysis

## Contexto
Este proyecto corresponde a un trabajo académico universitario en el que se aplican técnicas de análisis topológico de datos (TDA).

## Objetivo
- Introducir el uso de herramientas topológicas para el análisis de datos.
- Analizar la estructura de los datos mediante homología persistente.
- Interpretar los resultados obtenidos en términos del problema estudiado.

## Contenido
- Notebook principal con el desarrollo completo del análisis.
- Visualizaciones y conclusiones derivadas del uso de TDA.

## Herramientas
- Python
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebook
